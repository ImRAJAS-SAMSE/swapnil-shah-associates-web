# swapnil-shah-associates-web
Official full-stack digital presence for Swapnil Shah &amp; Associates. Built with React.js, Node.js, and hosted on secure AWS Architecture.

## Project Structure

```
swapnil-shah-associates-web/
├── client/          # React.js frontend application
├── server/          # Node.js/Express backend application
├── infrastructure/  # AWS/Terraform configurations and diagrams
└── docs/            # Internship reports and documentation
```

### Directory Overview

- **client/**: Contains the React.js frontend code for the web application
- **server/**: Contains the Node.js/Express backend API and server logic
- **infrastructure/**: Contains AWS infrastructure code, Terraform configurations, and architecture diagrams
- **docs/**: Contains internship reports, project documentation, and related diagrams

Each directory contains its own README.md with specific information about its contents and usage.
